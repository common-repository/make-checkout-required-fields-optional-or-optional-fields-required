You can make wocommerce checkout required fields optional or optional fields required.

== Description ==
Woocommerce Plugin Must be Installed. Tested upto woocommerce 8.2.0

You can make wocommerce checkout required fields as optional from dashboard.

You can make wocommerce checkout optional fields as required from dashboard.

Full control over checkout fields.

Make any checkout field required or optional as per your requirement.

You can also hide any optional field or make required field optional and hide.

[youtube https://www.youtube.com/watch?v=VtKyo5l5064]

== Installation ==
1.Just download and install like other wordpress plugins.
2.Goto "Checkout Fields Setting" from the dashboard admin menu
3.Set checkout fileds optional or required as per your requirement.
4.Save settings and see the result.


== Frequently Asked Questions ==

= How can i make chekout required field optional? =
Install and Activate plugin. 
Goto "Checkout Fields Setting" from the dashboard admin menu. 
Select "Optional" option from the dropdown in front of the field you want to make optional. 
Click on Save settings button and see the result. 

= How can i make chekout optional field required? =
Install and Activate plugin. 
Goto "Checkout Fields Setting" from the dashboard admin menu. 
Select "Required" option from the dropdown in front of the field you want to make required. 
Click on Save settings button and see the result. 

= How can i make chekout required field optional and hide? = 
Install and Activate plugin. 
Goto "Checkout Fields Setting" from the dashboard admin menu. 
Select "Optional and Hide" option from the dropdown in front of the field you want to make optional and hide. 
Click on Save settings button and see the result.
